# -_-
Определение расстояния до впереди идущего автомобиля на основе изображений

алгоритм, позволяющий определить дистанцию до впереди идущего автомобиля в режиме реального времени, используя для этого датасет фотографий автомобилей с разного расстояния. Впоследствии этот алгоритм может быть использован в системах навигации для предупреждения об опасном сближении и для контроля за соблюдением дистанции.
